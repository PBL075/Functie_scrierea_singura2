# Proiect: Gestiune Note

Acest proiect este o aplicație consolă scrisă în **C#** care ajută la calcularea mediei.

## Descriere
Codul folosește o funcție personalizată pentru a procesa datele.

## Funcționalități
* **Calcul Automat**: Calculează media aritmetică.
* **Organizare**: Menține codul curat în Main().
